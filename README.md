# CyberShield: Fake News Shooter 🚀

Projeto desenvolvido com foco total no edital de Recuperação de Educação Digital e IA (Ensino Médio). 

## 🎯 Objetivo do Tema
O portal serve como uma ferramenta de conscientização comunitária ativa para combater os impactos das **Deepfakes** e da **Desinformação automatizada por Inteligência Artificial**. A interface utiliza elementos gamificados (simulando um "shooter" espacial onde o usuário elimina notícias falsas) para engajar e educar cidadãos digitais.

## 🛠️ Tecnologias Utilizadas
- **HTML5 Semântico:** Estruturação limpa através das tags fundamentais `main`, `section`, `footer` e elementos de formulário (`form`, `input`, `label`, `button`).
- **CSS3 Avançado com Flexbox:** Alinhamentos dinâmicos, transições de hover suaves para guiar a atenção do utilizador, variáveis CSS (`:root`) e responsividade total via **Media Queries** testada para dispositivos móveis.
- **JavaScript (Manipulação do DOM):** Lógica interativa com processamento de variáveis, validação de respostas do quiz em tempo real e recurso de acessibilidade (Modo Claro/Escuro).
